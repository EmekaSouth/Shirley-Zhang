Skills

Adobe Animate

I use Adobe Animate to create interactive animations and motion graphics. I'm familiar with timeline control, keyframing, and symbol management, which allows me to design smooth 2D animations for web projects, videos, and creative presentations.

Figma

I use Figma to design and prototype simple web layouts and UI ideas before coding. I understand basic design principles and can turn my mockups into working websites.

Krita

I use Krita for digital art and graphic design. I'm comfortable with its brush tools, layers, and color management features, allowing me to create detailed illustrations, logos, and web graphics with precision.

Photoshop

I use Photoshop for photo editing, retouching, and graphic design. I'm experienced with tools like layers, masks, filters, and blending modes, allowing me to create clean, professional visuals for web and social media use.
